Por que devo aprender Java?

O Java vem evoluindo constantemente, grupos como o Java Noroeste buscam difundir a tecnologia até em regiões interioranas. Mas por que devo aprender Java?

Multiplataforma

Desenvolva para computadores, web e dispositivos portáteis usando apenas Java!

Robusto

Plataforma robusta e completa para projetos dos mais variados tamanhos.

Atualizações

Mantida por comunidades e empresas, o Java sempre tem novidades em lançamento.

Comunidade

Forte grupo de usuários espalhados pelo globo dispostos a ajudar a comunidade.

Frameworks

Vasta variedade de frameworks que facilitam o trabalho dos desenvolvedores.

De qualquer sistema...

Desenvolva de qualquer sistema operacional, para qualquer sistema operacional!

A comunidade Java Noroeste

O Java Noroeste foi fundado em 24 de outubro de 2004 na cidade de Votuporanga. Desde lá vem contribuindo com a difusão do Java no noroeste paulista.

Quem somos

Um grupo que gosta de falar e espalhar o Java pela região Noroeste de São Paulo. Temos como objetivo o incentivo a programação (em Java, claro!).

Cidades

Votuporanga, São José do Rio Preto, Fernandópolis, Mirassol, Presidente Prudente, Jales, Marília, Olímpia, Birigui, Barretos, Guaíra, Araçatuba, Tupã, Dracena, Valentim Gentil, Catanduva, entre outros municípios do noroeste paulista.

Lutando por uma comunidade melhor

Acreditamos que interagindo com empresas e universidades podemos mudar o panorama tecnológico de nossa região. Difundir o Java é também apoiar a prosperidade e inovação tecnológica.

Faça parte

Gosta de Java ou quer aprender mais sobre a linguagem? Quer fazer amigos e ajudar em projetos open source? Entre em contato através do formulário e venha fazer parte do Java Noroeste!

Galeria de fotos

Muitas pessoas já passaram pelo Java Noroeste! Diversos encontros foram realizados, estudos agendados e palestras organizadas.

Sobre Nós

Somos um grupo de desenvolvedores apaixonados por tecnologia Java! Temos como objetivo aprender e difundir o Java na região noroeste do estado de São Paulo.
